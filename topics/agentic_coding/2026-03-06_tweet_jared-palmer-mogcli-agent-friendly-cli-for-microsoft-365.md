# Jared Palmer — mogcli (agent-friendly CLI for Microsoft 365)

*Jared Palmer — @jaredpalmer*

## TL;DR

- Jared Palmer announced mogcli, an unofficial agent-friendly CLI for Microsoft 365.
- The repo wraps Microsoft Graph workloads (mail/calendar/contacts/groups/tasks/onedrive) behind scriptable commands.
- Design emphasis is automation-ready output (--json/--plain), profile-aware auth flows, and safer write ops via --dry-run.

## Highlights

- Supports both consumer MSA and enterprise Entra ID accounts.
- Auth supports delegated and enterprise app-only modes.
- Aims to be practical for agent/tool workflows by keeping CLI output stable and machine-friendly.
