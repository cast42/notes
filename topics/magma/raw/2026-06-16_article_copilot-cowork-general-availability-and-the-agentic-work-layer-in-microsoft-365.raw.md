Microsoft is moving Copilot Cowork from preview to general availability and positioning it as a full agentic work system rather than a chat assistant. The core claim is that Cowork can execute long-running, multi-tool business tasks end to end, grounded in Microsoft 365 context and wrapped in enterprise controls. Microsoft says adoption was fast, with more than half of the Fortune 500 using it during the three-month Frontier period, and highlights examples like spreadsheet editing with dependency mapping, cross-version comparison of nearly 4,000 files, and sales pipeline triage that compressed a week of review into a morning.
*"You define the work and Cowork runs it end-to-end and returns a completed result"*
The article emphasizes three differentiators: enterprise trust boundaries, multi-model execution, and usage-based cost control. Cowork is cloud-hosted, grounded by Work IQ, and kept within Microsoft 365 security and compliance layers, with features such as audit logs, eDiscovery, Insider Risk Management, sensitivity-label inheritance, and more governance tooling at launch. On pricing, Microsoft says Cowork is added on top of the Microsoft 365 Copilot subscription and billed per task in Copilot Credits, based on model use, context retrieval, tool calls, and runtime. It frames budgeting around light, medium, and heavy task types plus user personas, and says internal testing found Cowork to be 30 to 40% cheaper per prompt on average than a competing setup using the same underlying model family.
Microsoft is also using the launch to expand Cowork's product surface. General availability brings a dedicated toggle in the Copilot app, plugin support, browser use via Edge in Frontier, and new admin controls across control, visibility, and efficiency. Those include tenant, group, and user spending limits, customizable alerts, usage reporting, model choice in Frontier, and two payment options: pay-as-you-go or committed volume discounts. The strategic message is that Cowork is meant to become the agentic layer on top of Copilot, while a forthcoming "Cowork 1" model is supposed to reduce cost further for everyday workloads without locking customers to a single model provider.
*"Customers decide when Cowork turns on, who gets access, and how much can be spent."*
