# Operating Systems: Three Easy Pieces

## TL;DR

The book explains how operating systems share processors, manage memory, coordinate concurrent work, and store data. It groups the subject into virtualization, concurrency, and persistence.

The authors make the full book available online for free. The book also includes practical homework and projects.

## Key takeaways

- Virtualization lets each program work as if it controls the processor and memory.
- Concurrency explains how programs coordinate work that happens at the same time.
- Persistence covers storage devices, file systems, and recovery after failures.
- Small examples and exercises connect each concept to working systems.
