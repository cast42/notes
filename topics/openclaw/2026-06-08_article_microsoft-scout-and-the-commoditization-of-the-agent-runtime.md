# Microsoft Scout and the commoditization of the agent runtime

*The New Stack*

## TL;DR

- The article argues that Microsoft building Scout on OpenClaw signals that agent runtimes are being commoditized rather than becoming the main source of enterprise value.
- In this view, the durable product is the control plane around the runtime: managed identity, governance, policy enforcement, auditability, distribution, and hardware-backed containment.
- The piece frames OpenClaw as the shared substrate while Microsoft differentiates through Entra identities, Agent 365, Work IQ, and Microsoft Execution Containers.

## Highlights

- "The control plane is the product."
- Scout is presented as Microsoft’s first always-on enterprise agent running on OpenClaw.
- The article compares the runtime layer to Android: a free common base with value moving to layers above and below it.
- Microsoft Execution Containers are described as moving containment into the operating system as a runtime primitive.
- The startup takeaway is that building just the agent loop is probably a weak business compared with building the governed stack around it.
